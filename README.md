## Prepare workspace in Visual Studio Code
Visit on websiste -> [LINK](https://code.visualstudio.com/docs/cpp/config-mingw)

## Compile programme:

### Windows
* cmd:
    ```
    start.bat
    ```
    
* Powershell:
    ```
    ./start.bat
    ```

### Linux
```
./start.sh
```

## Run application:

```
./run.exe
```
